## 자동 수정이 해결 못 하는 이유

dev-harness(Codex)가 코드 수정으로 고칠 수 있는 건:
- 규칙 매칭 패턴 추가
- 파라미터 전달 버그
- 프롬프트 개선

고칠 수 없는 건:
- **외부 API가 없는 것** (날씨, SMS, 실시간 교통)
- **OAuth 미설정** (캘린더 쓰기, 메일 발송)
- **도구 자체의 한계** (크롤러로 동적 사이트 파싱 실패)

→ 다음 단계는 **외부 서비스 연결**이 최우선.
