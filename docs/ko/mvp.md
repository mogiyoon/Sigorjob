# MVP

## 현재 MVP 상태

이 프로젝트는 이제 "핵심 흐름이 실제로 동작하는 MVP" 단계에 들어와 있다.

현재 사용 경계:

- 로컬 데스크톱, 로컬 웹, CLI가 가장 마찰이 적은 경로다
- 원격 및 모바일은 Quick Tunnel 또는 정식 Cloudflare Tunnel을 사용할 수 있다
- 패키징된 데스크톱 앱은 최종 사용자용 `cloudflared`를 함께 포함할 수 있다
- 소스 기반 원격 워크플로는 여전히 호스트 머신의 `cloudflared`에 의존한다
- 소스 체크아웃은 일반적인 개발 도구와 의존성이 필요하다

이미 구현된 것:

- 명령 입력 -> Task 생성 -> Tool 실행 -> 결과 조회
- 규칙 기반 라우팅 + AI 폴백
- `file`, `shell`, `crawler`, `time`, `system_info` Tool
- 승인 필요 작업 분기
- cron 기반 스케줄 등록
- Cloudflare Tunnel 기반 원격 연결
- 모바일 WebView 래퍼
- GUI 없는 서버용 CLI

## 아직 남은 MVP 이후 작업

- WebSocket 기반 실시간 업데이트
- 더 정교한 오케스트레이션
- 추가 Tool (`calendar`, `message` 등)
- 모바일 위젯 실제 구현
- 배포/패키징 검증 마감

## 현재 완료 기준 관점

다음은 이미 충족했거나 대부분 충족한 항목이다.

- 규칙에 정의된 명령은 AI 없이 실행된다
- 모호한 명령은 AI가 해석 후 실행될 수 있다
- 위험한 작업은 승인 또는 차단 경로로 분기된다
- 실행 결과가 DB에 기록된다
- 프론트에서 명령 입력과 결과 확인이 가능하다
- CLI 환경에서도 같은 파이프라인을 사용할 수 있다
