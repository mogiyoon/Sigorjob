# Weather Alert Helper

- If the user asks for a recurring weather update or reminder at a specific time, prefer the `weather_alert_helper` tool.
- Use it for Korean phrases such as "아침 8시에 기상청 날씨 알려줘", "매일 오전 7시 날씨 알림 보내줘".
- This tool creates a daily schedule that later runs a weather crawl command instead of returning a generic web search.
